Crea y diseña tus propias páginas web

1. Instalamos el editor de código


En este video instalaremos un editor de código con el cúal comenzaremos a realizar nuestra página web, recuerda que no es necesario tener conocimientos previos ya que comenzaremos desde lo más básico y de una manera dinámica.

2. Introducción a
HTML5


El HTML es la estructura y la base de nuestra web, el primer archivo que descarga el navegador. Este archivo tiene el texto de nuestra web, y a la vez llama a otros archivos que van a ser necesarios para visualizarla, como CSS, JavaScript o imágenes, veámos como funciona.

3. Uso de etiquetas y atributos


Las etiquetas y atributos son fragmentos de código que permiten crear elementos, estructuras básicas del lenguaje de programación HTML en el que se escriben las páginas web porque es el que entienden los navegadores, veamos un poco más sobre estas.

4. Imágenes y
enlaces


Ahora veremos la primera forma de cargar contenido externo en nuestra web, con la etiqueta de imágen. Además , gracias a la etiqueta de enlace, podremos vincular a otras páginas o crear una navegación. ¡No te puedes perder este video!

5. Comenzamos con CSS


En esta unidad introducimos el uso de las hojas de estilo, aprederemos a vincular una hoja de estilos, la sintaxis, el funcionamiento básico y los estilos tipográficos

6. Estructuras en CSS


Vamos a conocer el modelo de caja de CSS y a empezar a crear estructuras a partir de las propiedades de layout fundamentales, como floats o position.

7. Decoración en CSS


Vamos a profundizar en las propiedades de CSS para decoración. En esta unidad vamos a ver los estilos para listas, bordes, esquinas redondeadas, sombras, fondos. y webfonts.

8. Técnicas responsive


Ahora que tenemos un conocimientos más sólidos sobre las propiedades de layout, vamos a ver cómo hacer una estructura responsive, capaz de adaptarse a múltiples tamaños de pantalla y condiciones.
